# Screenshot 应用商店截图制作工具

## 项目介绍

如果你看不懂这个App是做什么的，说明你还不是一名合格的App开发人员。

## 主要技术

- 美工妹子制作大量应用商店截图模板，然后标注好主标题、副标题、截图坐标、截图尺寸、附件坐标等参数。然后将这些数据整理成json数据，在应用中读取json，以面向对象方式来进行处理。

- 这个app没有什么技术难点，有的只有创意。
